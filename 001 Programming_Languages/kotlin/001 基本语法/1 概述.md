## 变量
使用`var` `val` 关键字来声明变量，后紧接变量名
var= variable 变量
val= value 值

### val
使用此关键词生命只赋予一次变量，不可变的只读局部变量，初始化后不可重现分配
```
val x:Int =5
```

### var
可以改变值的关键词
```
var x: Int = 5
// Reassigns a new value of 6 to the variable x
x += 1
```
---
kotlin 支持类型推导并自动识别声明变量的类型。故可以省略变量名后的类型

```
var x=5
```


可以在声明变量的同时初始化变量， 也可以在声明变量后再初始化，在第二种情况要指定类型
specify 指定/具体说明
```
// Initializes the variable x at the moment of declaration; type is not required
val x = 5
// Declares the variable c without initialization; type is required
val c: Int
// Initializes the variable c after declaration 
c = 3
// 5 
// 3
```


### 可以在顶层声明变量
```
val PI = 3.14
var x = 0

fun incrementX() {
    x += 1
}
```

```

```

## 条件表达式（conditional experssions）
if 类似c++
``
```
fun maxOf(a: Int, b: Int): Int {
    if (a > b) {
        return a
    } else {
        return b
    }
}
```

### 变种使用 可以作为表达式
```
fun maxOf(a: Int, b: Int) = if (a > b) a else b
```

##  when表达式
类似switch
```
fun describe(obj:Any): String=
	when(obj){
		1 ->"one"
		"hello" ->"Greeting"
		is Long ->"Long"
		!is String ->"Not a string"
		else ->"Unknown"
	}
```

## 类型检查与自动类型推导（automatic casts）
explicit 显式
implicit 隐式
检查表达式是否是某个类型的实例，如果检查了不可变的局部变量或属性，则不需要显示转换
```
fun getStringLength(obj: Any): Int? {

    if (obj is String) {

        // `obj` is automatically cast to `String` in this branch

        return obj.length

    }

​

    // `obj` is still of type `Any` outside of the type-checked branch

    return null

}
```
