Kotlin 中的字符串由 [`String`] 类型表示（represent）。

通常,字符串是一串双引号字符
```
val str = "abcd 123"
```

可以通过索引访问字符
```
for (c in str) {
    println(c)
}
```

字符串不可变，初始化后就不能更改它的值或赋予它新的值，所有转化字符串的操作都会在新对象中返回操作结果，原（original）字符串保持不变


```
val str = "abcd"

// Creates and prints a new String object
println(str.uppercase())
// ABCD

// The original string remains the same
println(str) 
// abcd
```

串接(concatenate)字符串请用加法，只要表达式的第一个元素是字符串，也可以用于串联其他类型的字符串

```
val s = "abc" + 1
println(s + "def")
// abc1def    
```


## 字符串字面量(String literals)

Kotlin 有两种字符串字面量类型：
- 转义字符串(Escaped strings)
- 多行字符串(Multiline strings)


### 转义字符串

可以包含转义字符
```
val s = "Hello, world!\n"
```



### 多行字符串


不含转义，可以包含换行和任意文本，以"""为界
```
val text = """
	 for (c in "foo")
		  print(c) 
		  """
```


使用 triMargin()去除前置空白
- 处理**多行字符串（`"""..."""`）**
- 默认去除每行开头的 **`|` 符号 + 左侧空白 **
- 自定义前缀符号，清理不规则缩进
- 保证代码编写时排版美观，**输出结果无多余缩进**

```
fun main() { val text = """ 
	|第一行 
	|第二行 
	|第三行 """.trimMargin() // 调用函数 
	
	println(text) }
```

```
第一行 
第二行 
第三行
```
会删除指定字符前的所有空白，保留右侧的内容


## 字符串模板

自动调用该表达式 返回结果 转换为字符串
在多行或转义字符串内使用 $
```
val i = 10
println("i = $i") 
// i = 10

val letters = listOf("a","b","c","d","e")
println("Letters: $letters") 
// Letters: [a, b, c, d, e]
```


可以使用${ }

```
val s = "abc"
println("$s.length is ${s.length}") 
// abc.length is 3
```


在多行字符串中使用要注意,单独表示$符号
```
val price = """ 
${'$'}_9.99 
"""
```
