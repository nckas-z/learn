## 函数的声明

使用 fun 来声明函数
括号（parentheses）内指定参数
返回值类型 可选

```
// 'double' is the name of the function
// 'x' is a parameter of Int type
// The expected return value is of Int type too
fun double(x: Int): Int {
    return 2 * x
}
```


## 函数使用

调用方法 类似c++
```
val result = double(2)
```


要使用成员函数或扩展函数要使用 .
```
// Creates an instance of the Stream class and calls read() Stream().read()
```

### 参数
使用 帕斯卡命名法(Pascal notation) 声明参数，`name: Type`
你必须用逗号(commas)分隔参数，并为每个参数明确赋予类型

```
	fun powerOf(number: Int, exponent:Int): Int{}
```

在函数体内部，接受实参（ arguments）为只读的，隐式声明为**val**
```
fun powerOf(number :Int, exponent: Int):Int {
	number
	number = 2 // Error: 'val' cannot be reassigned.
}
```

可以使用尾逗号（trailing comma）在声明函数形参时
```
fun powerOf(
	number:Int,
	exponent:Int,
){}
```

后置逗号有助于重构和代码维护(maintenance)：你可以在声明中移动参数，而不用担心哪个会是最后一个。


## 参数的默认值

同c++ 可以使用默认参数
形参带默认参数的就叫可选参数（ optional parameters.）

可选参数可以减少 多次重载（multiple overloads）

通过在参数声明后附加（appending）一个默认值来设置：`=`

```
fun read( b: ByteArray, // The default value of 'off' is 0 
		off: Int = 0, // The default value of 'len' is calculated // as the size of the 'b' 
		array len: Int = b.size, ) { /*...*/ }
```

默认参数后边的参数必须得显式指定值
```
fun greeting(
    userId: Int = 0,
    message: String,
) { /*...*/ }

fun main() {
    // Uses 0 as the default value for 'userId'
    greeting(message = "Hello!")
    
    // Error: No value passed for parameter 'userId'
    greeting("Hello!")
}
```

尾lambdas是例外，因为最后一个参数必须对应传递的参数
```
fun greeting(
    userId: Int = 0,
    message: () -> Unit,
)
{ println(userId)
    message() }

// Uses the default value for 'userId'
greeting() { println ("Hello!") }
// 0
// Hello!
```
