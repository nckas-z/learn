## 概念
- Android 官方**新一代 UI 框架**（对标 XML 布局 + View 体系）
- 完全用 Kotlin 编写，专门用于构建 Android 界面
- 属于 **Google Jetpack 库家族** 的一员


## Compose注解

Compose 函数是构建 UI 的核心。它们使用 @Composable 注解来标记，代表该函数是一个可组合的 UI 单元。
就是用这个注解的可以互相调用

```
@Composable  
fun MyApp(  
    modifier: Modifier = Modifier,  
    names: List<String> = listOf("World", "Compose")  
) {  
    Column(modifier = modifier.padding(vertical = 4.dp)) {  
        for (name in names) {  
            Greeting(name = name)  
        }  
    }  
}
```

## Preview注解

不用运行模拟器就可以观察Compose函数的效果，但缺点是不接受带参数的可组合函数

```
@Preview(showBackground = true)  
@Composable  
fun GreetingPreview() {  
    BasicsCodelabTheme {  
        MyApp()  
    }  
}
```


## 基础代码解析

```
class MainActivity : ComponentActivity() {  
    override fun onCreate(savedInstanceState: Bundle?) {  
        super.onCreate(savedInstanceState)  
        enableEdgeToEdge()  
        setContent {  
            BasicsCodelabTheme {  
                Scaffold(modifier = Modifier.fillMaxSize()) { innerPadding ->  
                    MyApp(  
                        modifier = Modifier.padding(innerPadding)  
                    )  
                }  
            }        }    }  
}
```
